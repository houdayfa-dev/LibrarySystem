## Overview

This project is a **console-based Library Management System written in C++**. It was built to practice and strengthen **core programming fundamentals** such as structs, enums, arrays, functions, and menu-driven programs.

The program allows users to enter book data and perform basic operations like listing books, finding the oldest book, counting available books, and filtering books by category.

---

## Features

* Add and store information for **5 books**
* Display **all books** with full details
* Find and display the **oldest published book**
* Count how many books are currently **available**
* Filter and display books by **category**
* Menu-driven interface with **input validation**

---

## Concepts Practiced

* `struct` for data modeling
* `enum` for menus and categories
* Arrays
* Functions and return values
* Input validation and control flow
* Switch statements
* Separation of logic into reusable functions

This project intentionally avoids advanced topics (OOP, STL containers, external libraries) to focus on **solid fundamentals**.

## How to Run the Program

### Requirements

* C++ compiler (GCC, Clang, or MSVC)

### Compile

```bash
g++ LibrarySystem.cpp -o LibrarySystem
```

### Run

```bash
./LibrarySystem
```

(On Windows)

```bash
LibrarySystem.exe
```
